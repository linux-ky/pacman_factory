## General info
Let's Make Unlimited Pacmen!
	
## Technologies
Project is created with:
* Node JS 16.14.2
* HTML 5
* CSS
	
## Setup
To run this project, clone this code to your local machine:
```
$ git clone https://github.com/linux-ky/pacman_factory
```

## How to Run
Load the index.html file into your browser.
* Press "Add PACMAN"
* Press "Start Game"
